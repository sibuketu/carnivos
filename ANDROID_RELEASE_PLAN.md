### 2. Android APK/AAB ビルド
```bash
cd "Downloads/CarnivOS/docs/primal-logic-app/primal-logic-web"
npm run build
npx cap sync android
cd android
./gradlew bundleRelease
```

### 3. 署名キー作成
```bash
keytool -genkey -v -keystore carnivos-release-key.jks -keyalg RSA -keysize 2048 -validity 10000 -alias carnivos
```
