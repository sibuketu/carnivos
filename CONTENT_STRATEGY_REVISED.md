## 🎯 基本戦略

### メインコンテンツ（90%）
教育・エンタメ系の価値提供動画
- 肉食/ケト/低糖質の正しい知識
- 栄養学の話
- 実践的なレシピ
- よくある誤解を解く

### テスター募集（10%）
- 10本に1本くらい専用動画
- または動画の最後5秒に「P.S. Beta募集中」

**理由**:
- テスター募集だけだと視聴者が離れる
- 有益なコンテンツで信頼を築く → 自然にテスター応募
- 長期的なファン獲得

## 🤖 OpusClip 使い方（簡易ガイド）

### 1. 動画アップロード
```
OpusClip ダッシュボード
→ "Upload Video" または "Paste YouTube Link"
→ ロング動画をアップロード
```

### 2. 設定
```
- Clip Length: 30-90秒
- Language: Japanese
- Add Captions: ON
- Aspect Ratio: 9:16（縦型）
```

### 3. AI処理待ち
```
処理時間: 10-20分（動画の長さによる）
→ メール通知
```

### 4. 結果確認
```
- 10-20本のショート候補が生成される
- 各クリップに「Viral Score」表示（0-100）
- スコア70以上を選択推奨
```

### 5. 編集（オプション）
```
- タイトル追加
- BGM調整
- トリミング微調整
```

### 6. ダウンロード or 自動投稿
```
- ダウンロード: MP4でダウンロード
- 自動投稿: TikTok/Instagram/YouTube に直接投稿
```
