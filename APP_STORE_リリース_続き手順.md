# Apple App Store リリース 続き手順

> **前回まで**: Apple Developer 加入、Bundle ID・App Store Connect 登録、メタデータ・スクショ・App Review 情報、証明書・Provisioning Profile・Codemagic ビルド（TestFlight/ビルド一覧へのアップロード）まで完了。  
> **このドキュメント**: プライバシー情報の完了 → アプリアイコン → ビルド選択 → 審査提出。  
> **ルール**: RULES 7.9。URL・貼る値はコードブロック。既にやった項目は書かない。

---

## やることの結論

未完了の項目だけ行う。**1 プライバシー情報を保存まで完了 → 2 アプリアイコン 1024×1024 をアップロード → 3 ビルドを審査用に追加 → 4 審査へ提出**の順。

---

## コピペ用（先に貼れる値）

**Bundle ID（参照用）**

```
com.CarnivOS.app
```

**App Store Connect（アプリ一覧から該当アプリを開く）**

```
https://appstoreconnect.apple.com/apps
```

**アプリのプライバシー（アプリ選択後。左メニュー「アプリのプライバシー」でも可）**

```
https://appstoreconnect.apple.com/access/privacy
```

※ 上記は「ユーザーとアクセス」内。アプリ単位のプライバシーは、アプリ選択 → 左「アプリのプライバシー」から。

---

## 1. プライバシー情報を完了する

| # | 操作 | 補足 |
|---|------|------|
| 1 | 次の URL を開く | App Store Connect でアプリを選択してから「アプリのプライバシー」を開く |
| 2 | データタイプを選択 | 健康・ユーザID・購入履歴・その他ユーザコンテンツなど、該当するものを選択。回答方針は `APP_STORE_PRIVACY_RULE.md` 参照。 |
| 3 | 各データの「個人情報に関連付くか」を回答 | 認証紐づきなら「はい」。 |
| 4 | 各データの「使用目的」を選択 | アプリの機能・製品のパーソナライズなど。広告・マーケティングに使っていなければチェックしない。 |
| 5 | 保存 | 画面の保存ボタンをクリック。 |

※ 先読み: データタイプを選ぶと、次の画面で「関連付くか」「使用目的」を聞かれる。

---

## 2. アプリアイコンをアップロードする

| # | 操作 | 補足 |
|---|------|------|
| 1 | App Store Connect で該当アプリ → iOS バージョン 1.0（または App 情報）を開く | アプリアイコン欄がある画面。 |
| 2 | 1024×1024px のアイコンをアップロード | PNG。既存は `docs/primal-logic-app/primal-logic-web/public` 内を確認。 |

※ アイコン欄の場所: 「iOSアプリ バージョン 1.0」を上にスクロールするか、「App情報」の「App Storeで表示」セクション。

---

## 3. ビルドを審査用に追加する

| # | 操作 | 補足 |
|---|------|------|
| 1 | App Store Connect で該当アプリ → 「iOSアプリ バージョン 1.0」を開く | ビルドセクションがある画面。 |
| 2 | 「ビルド」の「+」をクリック | アップロード済みビルド一覧が表示される。 |
| 3 | 表示されているビルドを 1 つ選択 | Codemagic でアップロードしたビルド。ビルドが無い場合は `IOS_CODEMAGIC_続き手順.md` のステップ 3 でビルド実行。 |
| 4 | 完了 | 審査用のビルドとして紐づく。 |

---

## 4. 審査へ提出する

| # | 操作 | 補足 |
|---|------|------|
| 1 | 同じ「iOSアプリ バージョン 1.0」画面で、提出準備完了の項目がすべて緑であることを確認 | プライバシー・アイコン・ビルドが済んでいれば「審査へ提出」が有効になる。 |
| 2 | 「審査へ提出」をクリック | 確認ダイアログが出たら送信。 |

---

## よくある状態と対処

| 現象 | 対処 |
|------|------|
| 「審査へ提出」が押せない | 提出準備完了の項目に未完了がある。プライバシー保存・アイコン・ビルド選択を確認。 |
| ビルドが一覧に無い | Codemagic でビルド実行し、Publishing → App Store Connect と API Key（.p8）を確認。`IOS_CODEMAGIC_続き手順.md` 参照。 |
