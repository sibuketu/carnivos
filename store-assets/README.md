# ストア用アセット（Play Console 4・5・6）

## 4 アプリのアイコン（512×512・1枚）

**採用: 2個目（青いリング＋ステーキ＋波形）。** pwa-512x512.png は使わない（ユーザー意志）。

| ファイル | 説明 |
|----------|------|
| **`app-icon-512.png`** | **2個目を512×512にリサイズ済み。これを「4 アプリのアイコン」にアップロードする。** |
| `icon-candidates-3in1.png` | 候補3案が1枚に並んだ画像（参考用）。 |

## 5 フィーチャー グラフィック（1024×500・横長）

**横長バナー。正方形のアイコン画像とは別。**

| ファイル | 用途 |
|----------|------|
| `store-feature-candidate-1.png` | 候補1（心電図・DNA・指紋等UI） |
| **`store-feature-adopted.png`** | **採用**（青いリング＋稲妻・炎）。**1024×500にリサイズ or トリミングしてからアップロード。** 中身はこの画像を横長に収めるか、アプリ名・ロゴ・ステーキ等を並べて新規作成。 |
| `store-feature-candidate-3.png` | 候補3（95%・72%ゲージ等） |

## 5・6の画像は後から変えられるか

**変えられます。**  
Play Console の「ストア設定」→「メインのストア掲載情報」で、フィーチャーグラフィック（5）と携帯電話版スクリーンショット（6）は**いつでも差し替え可能**です。新しい画像をアップロードして保存すれば反映されます（場合によっては審査のため反映に時間がかかりますが、機能としては後から変更可）。
